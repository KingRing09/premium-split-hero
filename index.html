‚ùì Troubleshooting

Images not showing?

Make sure you've uploaded images in the Theme Editor. Placeholder images show when no image is selected.

Animations not working?

Check if you have any CSS that might conflict with the section. The section uses the prefix hero- for all classes to avoid conflicts.

Mobile layout issues?

The section uses 768px as the breakpoint. If your theme uses different breakpoints, you can adjust this in the CSS.

Center content not fading?

The center content fades when hovering over split columns. This is intentional desktop behavior.
